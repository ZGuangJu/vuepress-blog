## var let 的区别

1. 局部作用域里，var 是函数作用域，let是块作用域
2. var 可以多次声明一个变量名，let不可以重复声明
3. let不会再作用域里被提升，var会提升
4. 再全局作用域中，let声明的变量不会成为window的属性，var声明的是window的属性

## const

1. 和let基本相同，但是声明时必须同时赋值
